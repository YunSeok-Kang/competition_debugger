Error 심층 분석

source: {{ source|upper }} user={{ user }}
이 페이지는 현재 보고 있는 submission을 기준으로 분석합니다. 아래에서 비교할 submission을 고른 뒤 `분석 갱신`을 누르면, `선택한 submission 비교` 탭에 비교 결과가 반영됩니다.
사용 순서
1. 분석 범위와 비교할 submission을 선택합니다.
2. `선택 내용으로 분석 갱신`을 누릅니다.
3. 현재 submission 결과는 `현재 submission 분석`, 비교 결과는 `선택한 submission 비교` 탭에서 확인합니다.
수동 태그 기반 특정 그룹
여기서 만든 그룹 조건은 아래 `현재 submission 분석`과 `선택한 submission 비교`에 동일하게 적용됩니다.
